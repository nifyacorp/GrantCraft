# GrantCraft

GrantCraft is a platform for AI-powered grant discovery and writing assistance.

## Components

This project is split into multiple repositories:

- **Frontend**: [GrantCraft-frontend](https://github.com/nifyacorp/GrantCraft-frontend)
- **Backend**: [GrantCraft-backend](https://github.com/nifyacorp/GrantCraft-backend)

## Deployment

See `cloud-run-deployment.md` for detailed deployment instructions.

## Architecture

- Frontend: Next.js, deployed on Netlify
- Backend: Python FastAPI, deployed on Google Cloud Run
- Database: MySQL, on Google Cloud SQL
