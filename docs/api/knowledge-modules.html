Knowledge Enhancement Modules

This document provides detailed information about the knowledge enhancement modules in GrantCraft. These modules provide researchers with resources, guidance, and best practices to improve their grant proposals.

Note: These modules are designed to be easily extensible with new agencies, best practices, and resources.

fundingAgencies Module

The fundingAgencies module provides information about funding agencies, their preferences, success criteria, and priorities. This helps researchers target appropriate funding sources and tailor their proposals accordingly.

Knowledge Module Agency Information

Constants

FUNDING_AGENCIES

An object containing detailed information about various funding agencies. Each agency entry includes properties like name, description, website, priorities, review criteria, and more.

Example Structure
{
  NSF: {
    id: 'nsf',
    name: 'National Science Foundation',
    acronym: 'NSF',
    description: '...',
    website: 'https://www.nsf.gov/',
    logo: '/assets/agencies/nsf-logo.png',
    primaryFocus: ['Science', 'Engineering', 'Education'],
    annualBudget: '$8.5 billion (2023)',
    typicalAwardSize: { ... },
    successRate: '20-25%',
    reviewCriteria: [ ... ],
    currentPriorities: [ ... ],
    applicationProcess: { ... },
    divisions: [ ... ],
    tips: [ ... ]
  },
  NIH: { ... },
  DOE: { ... },
  // other agencies
}

Functions

getAllAgencies()

Gets all funding agencies in the database with basic information.

Returns

Array of agency objects with basic information (id, name, acronym, description, primaryFocus, website)

Example
import { getAllAgencies } from '../lib/fundingAgencies';

const agencies = getAllAgencies();
console.log(`Found ${agencies.length} funding agencies`);
getAgencyDetails(agencyId)

Gets detailed information about a specific funding agency.

Parameters

Name Type Description
agencyId string The ID of the agency to retrieve (e.g., 'nsf', 'nih')

Returns

Detailed agency information object or null if not found

Example
import { getAgencyDetails } from '../lib/fundingAgencies';

const nsfDetails = getAgencyDetails('nsf');
if (nsfDetails) {
  console.log(`Success rate: ${nsfDetails.successRate}`);
  console.log(`Review criteria: ${nsfDetails.reviewCriteria.length}`);
}
searchAgencies(criteria = {})

Searches for agencies matching specific criteria.

Parameters

Name Type Description
criteria object Search criteria object
criteria.focus Array<string> Primary focus areas to match
criteria.priorities Array<string> Current priorities to match
criteria.keyword string Keyword to search in name and description

Returns

Array of matching agency objects

Example
import { searchAgencies } from '../lib/fundingAgencies';

// Search by focus areas
const scienceAgencies = searchAgencies({ focus: ['Science'] });

// Search by priorities
const aiAgencies = searchAgencies({ priorities: ['Artificial intelligence'] });

// Search by keyword
const energyAgencies = searchAgencies({ keyword: 'energy' });

// Combined search
const results = searchAgencies({
  focus: ['Engineering'],
  priorities: ['Climate'],
  keyword: 'sustainability'
});
getRecommendedAgencies(topic)

Gets a list of funding agencies that match a research topic, sorted by relevance score.

Parameters

Name Type Description
topic string Research topic to match against agencies

Returns

Array of recommended agencies with relevance scores, sorted by score

Example
import { getRecommendedAgencies } from '../lib/fundingAgencies';

const topic = 'Climate change mitigation through carbon capture technology';
const recommendations = getRecommendedAgencies(topic);

console.log(`Top recommendation: ${recommendations[0].name}`);
console.log(`Relevance score: ${recommendations[0].relevanceScore}`);

bestPractices Module

The bestPractices module provides grant writing best practices, examples, and advice tailored to different research disciplines, grant types, and funding agencies.

Knowledge Module Grant Writing Guidance

Constants

GENERAL_BEST_PRACTICES

Collection of general best practices applicable to most grant proposals.

FIELD_SPECIFIC_PRACTICES

Field-specific best practices organized by research discipline (biomedical, physical_sciences, social_sciences, humanities, engineering, interdisciplinary).

AGENCY_SPECIFIC_PRACTICES

Agency-specific best practices tailored to specific funding organizations (NSF, NIH, DOD, DOE, EU).

Example Practice Structure
{
  id: 'clear-writing',
  title: 'Use Clear, Concise Writing',
  description: 'Reviewers read many proposals in a short time. Clear, straightforward language helps your ideas stand out.',
  tips: [
    'Use active voice and present tense when possible',
    'Avoid jargon or technical terms without explanation',
    '...'
  ],
  example: 'Instead of "Cellular proliferation will be analyzed using flow cytometric methods", write "We will use flow cytometry to analyze cell growth."',
  category: 'writing',
  relevance: ['all']
}

Functions

getAllBestPractices()

Gets all general best practices.

Returns

Array of general best practice objects

Example
import { getAllBestPractices } from '../lib/bestPractices';

const practices = getAllBestPractices();
console.log(`Found ${practices.length} general best practices`);
getFieldPractices(field)

Gets field-specific best practices for a particular research discipline.

Parameters

Name Type Description
field string The research field (e.g., 'biomedical', 'physical_sciences')

Returns

Array of field-specific best practice objects

Example
import { getFieldPractices } from '../lib/bestPractices';

const biomedicalPractices = getFieldPractices('biomedical');
biomedicalPractices.forEach(practice => {
  console.log(`- ${practice.title}`);
});
getAgencyPractices(agency)

Gets agency-specific best practices for a particular funding agency.

Parameters

Name Type Description
agency string The agency ID (e.g., 'NSF', 'NIH')

Returns

Array of agency-specific best practice objects

Example
import { getAgencyPractices } from '../lib/bestPractices';

const nsfPractices = getAgencyPractices('NSF');
nsfPractices.forEach(practice => {
  console.log(`- ${practice.title}`);
});
getRecommendedPractices(options = {})

Gets recommended best practices based on research field, target funding agency, and/or category.

Parameters

Name Type Description
options object Options for retrieving recommendations
options.field string Research field
options.agency string Target funding agency
options.category string Practice category (e.g., 'impact', 'methodology')

Returns

Array of recommended best practices without duplicates

Example
import { getRecommendedPractices } from '../lib/bestPractices';

// Get practices for biomedical research targeting NIH
const practices = getRecommendedPractices({
  field: 'biomedical',
  agency: 'NIH'
});

// Get methodology practices for engineering
const methodologyPractices = getRecommendedPractices({
  field: 'engineering',
  category: 'methodology'
});
searchPractices(keyword)

Searches best practices by keyword in title or description.

Parameters

Name Type Description
keyword string Keyword to search for

Returns

Array of matching best practice objects

Example
import { searchPractices } from '../lib/bestPractices';

// Search for practices about budget
const budgetPractices = searchPractices('budget');

// Search for practices about methodology
const methodologyPractices = searchPractices('methodology');

KnowledgeHub Component

The KnowledgeHub component is the main entry point for all knowledge enhancement features. It provides a unified interface for accessing funding agencies, best practices, and other resources.

Component Knowledge Enhancement

Props

Name Type Required Description
projectInfo object No Information about the current project for personalized recommendations
onClose function Yes Callback function to close the KnowledgeHub

Features

  • Agency database browsing and searching
  • Best practices library
  • Additional resources and links
  • Project-specific recommendations
Example Usage
import KnowledgeHub from '../components/KnowledgeHub';

const MyComponent = () => {
  const [showKnowledgeHub, setShowKnowledgeHub] = useState(false);
  const projectInfo = {
    title: 'Climate Change Research',
    topic: 'Carbon capture technologies'
  };
  
  return (
    

      
      
      {showKnowledgeHub && (
         setShowKnowledgeHub(false)}
        />
      )}
    

  );
};

AgencyProfile Component

The AgencyProfile component displays detailed information about a funding agency, including review criteria, application process, and expert tips.

Component Agency Details

Props

Name Type Required Description
agency object Yes Agency object with detailed information
onClose function Yes Callback function to close the profile

Features

  • Overview tab with general agency information
  • Funding tab with award sizes and application process
  • Review Criteria tab with detailed review information
  • Tips & Divisions tab with expert advice and organizational structure
Example Usage
import AgencyProfile from '../components/AgencyProfile';
import { getAgencyDetails } from '../lib/fundingAgencies';

const MyComponent = () => {
  const [selectedAgency, setSelectedAgency] = useState(null);
  const [showProfile, setShowProfile] = useState(false);
  
  const handleOpenProfile = (agencyId) => {
    const agencyDetails = getAgencyDetails(agencyId);
    setSelectedAgency(agencyDetails);
    setShowProfile(true);
  };
  
  return (
    

      
      
      {showProfile && selectedAgency && (
         setShowProfile(false)}
        />
      )}
    

  );
};
